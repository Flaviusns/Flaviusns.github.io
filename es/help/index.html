Sección de ayuda

En esta sección vamos a comentar los problemas más comunes y como solucionarlos:

Archivos inválidos

Si la aplicación devuelve una alerta con la información sobre archivos inválidos, puede ser por múltiples razones debido a tus archivos.

Sigue leyendo para más información:

Formato de archivo erróneo

Asegurate de que el archivo comprimido utiliza un formato de compresión válido e incluye la extensión correcta. Las extensiones validas son

archivos .cbz, archivos .cbr, archivos .cb7. También asegurate de que el archivo descomprimido sigue la misma estructura:

La carpeta raíz contiene el nombre del comic y dentro de esta carpeta tienes directamente las páginas del cómic.

Para poder comprobar esto, puedes descomprimir tu cbz,cbr,cb7 usando Keka

Asegurate de que el nombre del cómic es el mismo que la carpeta raíz.

Una vez que tengas el cómic con la estructura correcta, puedes comprimir el comic utilizando un formato diferente, el mejor formato para la app es
el archivo cbz.

Para comprimir un cómic con el formato cbz, seleccióna la carpeta raíz de tu cómic y comprimela con el formato zip. Después, cambia la extensión de .zip a .cbz y después envíalo a tu dispositivo.

Ejemplo: MySuperComicRenamed.zip -> MySuperComicRenamed.cbz

Nombre erróneo

Otro error común es tener el cómic con un nombre distinto al original.
Descomprime tu cómic utilizando Keka y asegurate de que el nombre del cómic es el mismo que el de la carpeta raíz.

Por ejemplo, si el nombre del cómic es MySuperComic.cbz. Cuando lo descomprimas, la carpeta raíz debería llamarse MySuperComic.

Si el archivo raíz tiene otro nombre, por favor cambialo y vuelve a comprimirlo a zip, y después cambiale la extensión a cbz.

Ejemplo: MySuperComicRenamed.zip -> MySuperComicRenamed.cbz

¿Por qué la portada no es la primera página?

Si tienes un cómic cuya portada no es la imagen que aparece en la colección, por favor sigue los siguientes pasos:

Descomprime el archivo

Abre la carpeta raíz y comprueba si la portada es la primera imagen si ordenas el contenido por nombre

Si no, cambia el nombre de la portada para que sea la primera en la carpeta.

Comprime de nuevo el archivo a zip y cambia la extensión a cbz.
